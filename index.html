Tips and Tricks

Basic tips 
/*----u can put you URI's in your code without compilation error----*/
   #include <iostream>
   int main() {
   	using namespace std;
   	http://www.google.com
   	int x=5;
   	cout << x;
}

Explanation:

Any identifier followed by a : becomes a (goto) label in C++.

Anything followed by // becomes a comment so in the code above, http is a label and //google.com/ is a comment.

The compiler might throw a warning however, since the label is unutilized.
